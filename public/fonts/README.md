# Police Aeonik

Placez ici les fichiers de police Aeonik dans les formats suivants :

- `Aeonik-Regular.woff2` (format recommandé, meilleure compression)
- `Aeonik-Regular.woff` (fallback)
- `Aeonik-Regular.ttf` (fallback)
- `Aeonik-Medium.woff2`
- `Aeonik-Medium.woff`
- `Aeonik-Medium.ttf`
- `Aeonik-Bold.woff2`
- `Aeonik-Bold.woff`
- `Aeonik-Bold.ttf`

Les déclarations `@font-face` sont déjà configurées dans `src/index.css`.

**Note** : Aeonik est une police premium. Assurez-vous d'avoir les droits d'utilisation pour le web.
